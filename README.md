## 为啥大家都在聊北京9929线路？

如果你对国内网络有点了解，"AS9929"这个词不陌生——这是联通的精品A网，跟普通的AS4837民用网相比，延迟低、稳定性强，晚高峰不容易被QoS限速。很多需要稳定海外出口的用户，对这条线路情有独钟。

酷雪云在北京有自己的数据中心，推出了基于9929线路的VPS系列，主要特点是：

- **电信联通双优化入口**：电信和联通用户出海走AS9929专线，跨运营商、晚高峰都能保持稳定
- **双独立IPv6**：每个套餐配两个独立IPv6地址，独立带宽，对有IPv6需求的用户很友好
- **Ryzen 7000系列处理器**：从第二档套餐起，用的是AMD Ryzen 7000系列，性能比入门档强不少
- **晚高峰测试稳定**：有用户晒过晚高峰的IPERF3测速，基本能跑满带宽，丢包率低

当然，这条线路对电信联通用户更友好，移动用户的体验相对一般，这是9929的特性，不是酷雪云的锅。
